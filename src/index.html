Forza Horizon 5 Custom HUD

By Paburrito

Controls and Hotkeys

  • Drag titlebar - Move window
  • Ctrl+L - Lock/unlock position & enable click-through
  • Ctrl+S - Save custom position
  • Ctrl+Shift+R - Restore custom position
  • Ctrl+R - Force re-learn rev limiter for current car
  • Ctrl+Alt+R - Quick restart the HUD

How to use the HUD

  • Just make sure that Forza Horizon 5 "Data out" is activated
  • Data out IP Address: 127.0.0.1
  • Data out IP Port: 5300
  • That's it! You're ready to use the HUD with the game

Auto-Learning Rev Limiter

The HUD automatically learns each car's rev limiter:

  • Bounce off the limiter for about 2-3 seconds while driving, you'll get the best readings by using at least manual transmission
  • Red zone will appear at the correct RPM
  • Learns once per session, this is so it doesn't interfere mid race
  • This info is stored internally, so if you switch to a previously learned car, it can recall it instantly
  • You can always force the HUD to re-learn your current car's rev limiter with Ctrl+R in case it doesn't match properly

Auto-Learning HP/TQ/Boost gauges

Gauges adapt to your car's power output:

  • Resets when you change cars or tunes
  • Learns max values as you drive
  • Needles move upwards when accelerating, and into the red zone when decelerating (negative HP and Torque)

Window Positioning

  • Unlock to see window boundaries (dashed border, useful for initial positioning on screen)
  • Auto-saves position/size when you click on the lock icon
  • Remembers the last size and position when re-opening the app
  • Can custom save / restore size and position in case you accidentally moved the HUD out of place

Features

  • Transparent overlay (OBS-compatible, just make sure to use Window Capture and Capture Method: Windows 10 (1903 and up))
  • When you lock the window, it becomes click-through so it doesn't interfere with the game
  • The app is not limited to a singular screen, so you can move this to the screen you're playing on

Personal notes

  • Initially this whole thing started because I made a custom skin for GamepadViewer, then I realized I could make a HUD using Forza's telemetry export feature
  • After testing stuff around, I discovered I could mix my love for Forza Horizon 5 and Need for Speed: Underground 2 to make a custom HUD with both styles in mind
  • What you see is the result of it, and the best of all, like Todd Howard says: "It just works", only that this does it and being lightweight as well while at it
  • I hope you get to enjoy this HUD as much as it was fun for me while developing it
